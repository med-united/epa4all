How is this header computed?
  1. Build a FHIR Organization resource carrying the SMC-B Telematik-ID and the practice name.
  2. Serialize to JSON and UTF-8 encode the bytes.
  3. Base64-encode the UTF-8 bytes and send the result as the X-Requesting-Organization request header.

FHIR Organization payload

{
  "resourceType": "Organization",
  "identifier": [{
    "system": "https://gematik.de/fhir/sid/telematik-id",
    "value": "<SMC-B Telematik-ID>"
  }],
  "name": "<Organization Name>"
}

Shell equivalent (Ubuntu defaults — printf + base64)

TID="1-SMC-B-Testkarte-883110000141773"
ORG="Praxis Lukas Freiherr Hünsch TEST-ONLY"

printf '{"resourceType":"Organization","identifier":[{"system":"https://gematik.de/fhir/sid/telematik-id","value":"%s"}],"name":"%s"}' \
  "$TID" "$ORG" | base64 -w0

Recomputes automatically when SMC-B Telematik-ID or Organization Name changes above.
